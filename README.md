# Boston Dynmaics Spot Implementation for LeRobot

LeRobot implementation of the Boston Dynamics Spot with Arm.

Only works with LeRobot version <= 4.4 (since newer versions use pyhton 3.12 --> not compatible with Spot-Api)


## Dependencies

Teleoperator can be found here:

https://github.com/J0STR/lerobot_teleoperator_spot

## Install

Clone Repository in LeRobot-Workspace.

Move into the folder:

```bash
cd lerobot_robot_spot
```

Install:

```bash
pip install e .
```
